# Handiloom-wood

A modern and responsive furniture eCommerce website built with **Next.js** and managed with **Hygraph CMS**.  
Designed and developed by **Preet Tech OPC Private Limited**, this project focuses on a smooth shopping experience, elegant UI, and high performance across all devices.

## Features

- Built with Next.js
- Responsive and mobile-friendly design
- Product listing and showcase sections
- CMS-powered content management with Hygraph
- Clean and modern furniture store UI
- Fast loading and SEO-friendly structure

## Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS
- **CMS:** Hygraph
- **Deployment:** Vercel
